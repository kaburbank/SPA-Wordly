# SPA-Wordly
Summative Lab for SE03-PT-01 // Front-End Development: Javascript Interactions- Single Page Application - Wordly

Wordly Dictionary Search is an interactive dictionary search website application sourced from Free Dictionary API. The Single Page Application was built using Javascript, HTML, and CSS.  Features include:
A dictionary word search,
a definitions list with parts of speech and examples,
a synonyms list (if available) with a click to search feature,
audio pronunciation playback button.


To use the application, simply clone the repository and open index.html in your web browser or local server. Then, enter a word into the search bar and click the "Search" button. 

Features implemented in this applicaiton include form submission handling, API integration, event listeners, input validation, and error handling.
