# SPA-Wordly
Summative Lab for SE03-PT-01 // Front-End Development: Javascript Interactions- Single Page Application - Wordly

Wordly Dictionary Search is an interactive dictionary search website application sourced from Free Dictionary API. The Single Page Application was build using Javascript, HTML, and CSS.  Features include:
A dictionary word search,
Definitions with parts, of speech and examples,
Synonyms list if available,
Audio pronunciation playback button.


To use the application, simply clone the repository and open index.html in your web browser or local server. Then, enter a word into the search bar and click the "Search" button.
